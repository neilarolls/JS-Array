WELCOME
This project displays random pictures from Lorem Picsum.

It keeps a list of email addresses that you have entered and allows you to assign any number of images to each email address.

The images attached to each address are displayed in the gallery below. There are options to select the size and orientation of the current random image before assigning it.

This project demonstrates the use of arrays in JavaScript.

To begin, enter a valid email address below.

( Note: please do not enter live email addresses into this demo! )
