WELCOME
This web page displays random images from Lorem Picsum.

It keeps a list of email addresses that you have entered and allows you to assign any number of images to each email address.

The image panel displays the current random image with a button to toggle greyscale, a blur level slider and a button to assign the image to the currently selected email address. Images attached to each address are displayed in the gallery.

This project demonstrates the use of arrays in JavaScript.

To begin, enter a valid email address below.
